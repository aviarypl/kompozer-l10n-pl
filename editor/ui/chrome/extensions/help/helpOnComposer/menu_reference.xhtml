Rozmiar tekstu

Mniejszy

Zmniejsza wielkość pokazywanego tekstu. Aby przywrócić oryginalny rozmiar, należy nacisnąć klawisze Ctrl+0.

Większy

Zwiększa rozmiar wyświetlanego tekstu. Rozmiar tekstu można także zmienić kręcąc rolką myszy przy wciśniętym klawiszu Ctrl lub za pomocą skrótów Ctrl++ i Ctrl+-.

Kodowanie znaków
